# Лабораторная работа: Фабричный метод (Factory Method)

## 📌 Аннотация

В этой лабораторной работе вы познакомитесь с **порождающим паттерном проектирования «Фабричный метод»**. Вам предстоит реализовать систему планирования космических экспедиций, где разные типы миссий создаются через специализированные фабрики.

На примере исследовательских, боевых и грузовых миссий вы увидите, как фабричный метод позволяет:

- Отвязать клиентский код от конкретных классов продуктов.
- Легко добавлять новые типы миссий без изменения существующего кода.
- Соблюдать принцип открытости/закрытости (OCP).

## 🎯 Цель работы

Освоить паттерн «Фабричный метод» на практике. Научиться создавать иерархии продуктов и создателей, а также расширять систему новыми типами объектов без модификации существующего кода.
